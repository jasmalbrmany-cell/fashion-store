إذا رأيت "pnpm install":
  → لم تغيّر Install Command في الخطوة 2
  → ارجع وغيّره إلى "npm install"

إذا رأيت "VITE_SUPABASE_URL is not defined":
  → لم تضف Environment Variables
  → ارجع للخطوة 2 وأضف جميع المتغيرات

إذا رأيت "Build failed":
  → تحقق من Build Command
  → يجب أن يكون "npm run build" بالضبط
