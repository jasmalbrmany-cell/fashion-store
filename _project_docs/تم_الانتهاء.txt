📋 قائمة التحقق:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

الخطوة 1 - Supabase:
  □ فتحت رابط Supabase API Settings
  □ ضغطت "Reset" بجانب anon key
  □ نسخت المفتاح الجديد

الخطوة 2 - Vercel Settings:
  □ غيّرت Framework Preset إلى: Other
  □ غيّرت Build Command إلى: npm run build
  □ غيّرت Install Command إلى: npm install
  □ أضفت VITE_SUPABASE_URL
  □ أضفت VITE_SUPABASE_ANON_KEY (الجديد)
  □ أضفت VITE_APP_NAME
  □ أضفت VITE_WHATSAPP_DEFAULT
  □ أضفت FIRECRAWL_API_KEY (اختياري)
  □ حددت Production + Preview + Development لكل متغير
  □ ضغطت Save

الخطوة 3 - Redeploy:
  □ فتحت صفحة Deployments
  □ ضغطت على أحدث deployment
  □ ضغطت ⋯ → Redeploy
  □ ألغيت تحديد "Use existing Build Cache"
  □ ضغطت Redeploy
  □ انتظرت حتى انتهى البناء
