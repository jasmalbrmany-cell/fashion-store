## 🐛 استكشاف الأخطاء

### المشكلة: "فشل من جميع المصادر"
**الأسباب المحتملة**:
- الموقع محمي بـ Cloudflare
- الرابط غير صحيح
- المنتج محذوف

**الحل**:
- جرب رابط آخر
- تأكد من الرابط
- استورد يدوياً

### المشكلة: "فشل رفع الصور"
**الأسباب**:
- Supabase Storage غير مُعد
- صلاحيات خاطئة

**الحل**:
- شغّل `setup_storage.sql`
- تحقق من RLS policies

### المشكلة: "التصنيف الذكي لا يعمل"
**السبب**: Gemini API Key غير موجود

**الحل**:
- أضف `VITE_GEMINI_API_KEY` في Vercel
- أعد النشر
